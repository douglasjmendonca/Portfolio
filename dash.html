Detalhes do Projeto

O projeto teve como objetivo o desenvolvimento de um dashboard interativo para analisar os preços da gasolina no Brasil de 2004 a 2021. Foram utilizados os pacotes Python Streamlit, Plotly Express, Plotly Graph Objects e Pandas para criar visualizações dinâmicas. O dashboard apresenta tendências temporais, variações mensais, análises regionais e comparações por estado. O objetivo é identificar tendências de longo prazo, flutuações sazonais e variações geográficas nos preços da gasolina, auxiliando consumidores e tomadores de decisão. Os dados foram obtidos do conjunto "Gas Prices in Brazil" no Kaggle, compilados por Matheus Freitag. Explore o dashboard interativo para insights valiosos sobre os preços da gasolina no Brasil e tomada de decisão informada.
