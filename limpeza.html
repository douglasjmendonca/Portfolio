Detalhes do Projeto

Neste projeto, abordei algumas estratégias de limpeza de dados para uma empresa de segurança na área de telecomunicações. Os dados utilizados foram obtidos do conjunto "Telco Customer Churn" disponível no Kaggle, que foram ajustados especificamente para os objetivos deste projeto que é utilizar o Python para realizar limpeza e preparação de dados. A empresa possui um histórico de 150 mil registros, os quais serão utilizados para modelagem preditiva. Contudo, antes de avançarmos para a modelagem, é crucial realizar um trabalho minucioso de limpeza, uma vez que os dados apresentam diversos problemas.
